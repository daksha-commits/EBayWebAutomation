# EBay web automation using selenium, mocha and chai
This repo is part of my assignment for website automation using page object model design pattern in javascript
It is developed using mocha framework and chai assertion

## To start
- Access the repo branch from github or zip
- Clone the branch if from repo or unzip if from zip
- Run `npm i`

## Run Tests
- `npm test`

## Test report are generated under:
./ebayreport/reportsummary.html
